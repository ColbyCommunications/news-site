Connect your WordPress website with Cludo.

== Description ==

Automatically keep WordPress in sync with Cludo.

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload the `cludo-wordpress` folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
